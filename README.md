# 🧠 Artificial Neural Network (ANN)

Artificial Neural Networks are inspired by the human brain and consist of multiple layers of interconnected neurons.

### ANN Workflow

```
Input Layer
      ↓
Hidden Layer
      ↓
Hidden Layer
      ↓
Output Layer
```

### Applications

* Customer Churn Prediction
* Loan Approval
* Disease Prediction
* Sales Forecasting
* Credit Risk Analysis

---

# 🖼️ Convolutional Neural Network (CNN)

CNN is a specialized deep learning architecture mainly used for image processing and computer vision tasks.

### CNN Workflow

```
Input Image
      ↓
Convolution Layer
      ↓
ReLU Activation
      ↓
Pooling Layer
      ↓
Flatten Layer
      ↓
Fully Connected Layer
      ↓
Output
```

### Applications

* Image Classification
* Face Recognition
* Medical Imaging
* Object Detection
* Handwritten Digit Recognition
